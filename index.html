🔑 如何免費申請 Gemini API 金鑰

4 個步驟，大約 2 分鐘完成，完全免費。

  1. 登入 Google 帳號
    用您常用的 Google 帳號（例如學校 Gmail 或個人信箱）登入即可。若尚未登入，點下方藍色按鈕時系統會自動引導您登入。
  2. 前往 Google AI Studio
    點下方「開啟 Google AI Studio」按鈕，會在新分頁打開 aistudio.google.com。首次進入時可能會請您同意使用條款，勾選後按「Continue」即可。
  3. 點選「Create API key」建立金鑰
    進入網頁後，點選右上方的藍色按鈕 Create API key →「Create API key in new project」（建立於新專案），系統會自動產生一組金鑰。
    💡 如果畫面是英文，看到 Get API key 也是同一顆按鈕。
  4. 複製金鑰貼回本工具
    點金鑰右側的 📋 複製 圖示，回到本工具的「API 金鑰」欄位貼上 → 按「儲存設定」→ 再按「🔍 測試 API 金鑰」確認可用即大功告成！
    ⚠️ 重要提醒：金鑰等同於密碼，請勿分享或截圖外流。本工具僅儲存在您的瀏覽器本機（localStorage），不會上傳任何伺服器。
