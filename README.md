# 桃園市115學年度國民中學課程計畫 AI 審查工具

> 依據桃園市教育局115學年度國中審查標準（0505版審查文件），上傳課程計畫 PDF 即可透過 AI 自動審查 | 阿凱老師製作

🔗 **線上使用：** [https://cagoooo.github.io/JHScurriculum/](https://cagoooo.github.io/JHScurriculum/)
🏫 **國小版（1-6 年級）：** [https://cagoooo.github.io/curriculum/](https://cagoooo.github.io/curriculum/)

## 功能特色

- **🎨 編輯排版風 UI（v1.1.0 新版）**：橄欖綠土地系配色 + Source Serif 4 襯線標題；頂部 5 步驟條會跟著操作即時亮起進度；版面置中 1000px、雜誌感卡片+左側狀態色條,閱讀更舒適
- **📄 PDF 上傳擷取**：支援拖拉或點擊上傳，自動擷取課程計畫文字內容
- **📋 40+ 項國中審查標準內嵌**：涵蓋 0-1 到 7-11 全部項次（七至九年級），無需另外複製提示詞
- **✏️ 可彈性編輯提示詞**：如實際門數與規定不同（如5門→3門），可直接修改再送出
- **🤖 AI 自動審查**：整合 Google Gemini API，一鍵送出即可得到審查建議
- **📋 複製提示詞**：無 API 金鑰時，可複製完整提示詞到 ChatGPT / Gemini 網頁版使用
- **🔄 修正後重新審查**：修正 PDF 後可快速重新上傳，重複審查直到通過
- **⚠️ 特殊班級提醒**：選擇特教/藝才/體育班項次時，自動顯示需搭配人工審查提示
- **🎓 國中專屬**：所有提示詞已調整為第四學習階段（Ⅳ）、7-9 年級節數規定、國中專屬科目（歷史/地理/公民/生物/家政/童軍/輔導/資訊科技/生活科技）

## 使用方式

### 快速開始（線上版）

1. 前往 [https://cagoooo.github.io/JHScurriculum/](https://cagoooo.github.io/JHScurriculum/)
2. 展開「**API 設定**」，點擊「🔑 免費申請 Gemini API 金鑰」取得金鑰
3. 貼上 API 金鑰並儲存
4. **Step 1**：上傳課程計畫 PDF
5. **Step 2**：選擇審查分類與項次（可直接編輯提示詞調整條件）
6. **Step 3**：點擊「送出 AI 審查」，等待約 30–60 秒取得結果

### 申請 Gemini API 金鑰（免費）

1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey)
2. 登入 Google 帳號
3. 點擊「Create API key」
4. 複製金鑰，貼到工具的「API 金鑰」欄位

### 無 API 金鑰時

點擊「**複製提示詞**」，再將複製的內容貼到 [ChatGPT](https://chat.openai.com) 或 [Gemini](https://gemini.google.com) 網頁版使用。

## 注意事項

- **AI 審查僅供輔助參考**，偶爾可能有疏漏，正式審查結果以教育局委員審定為準
- 支援**文字型 PDF**；掃描版（圖檔）PDF 無法擷取文字
- **特殊教育班、藝術才能班、體育班**等特殊班級目前尚無完整專屬 Prompt，建議搭配人工審查
- **7-3 全校教師授課節數一覽表**：依據「桃園市國民中學教師授課節數實施要點 (114.09.12 修訂)」進行審查（已使用國中專屬要點）
- API 金鑰僅儲存在瀏覽器本機（localStorage），不會上傳至任何伺服器
- 如需調整審查條件（例如課程門數合併），可直接在「審查提示詞」欄位修改後送出

## 技術說明

- **純前端**單一 HTML 檔，無後端、無伺服器

- 程式架構承襲自國小版審查工具，提示詞與節數規定完全替換為國中版
